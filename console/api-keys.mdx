---
title: API Keys
description: Create and manage keys in the Console.
---

[Console → API Keys](https://apisale.ai/console/api-keys)

Each key includes:

| Setting | Default | Description |
|---------|---------|-------------|
| `rate_limit_per_minute` | 120 | Requests per minute |
| `max_concurrency` | 10 | Parallel runs for this key |

The keys table shows live **in flight** counts when runs are active.

## Create a key

1. Enter a name (e.g. "Production")
2. Choose `live` or `test`
3. Copy the secret immediately — it is not shown again
